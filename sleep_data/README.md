## Sleep Data Dataset (Kaggle)

This was a small initial analysis of a simple dataset on mammalian sleep patterns. I applied a linear regression model to predict sleep exposure based on biological features. The model performed poorly, indicating that the available features had limited predictive power.

**Key learnings:**
- Basic data manipulation with pandas
- Detecting missing or invalid numerical values
- Using scatter plots and correlation analysis before model building
- Understanding that poorly motivated or artificial features are often not predictive
