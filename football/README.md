# Football Match Analysis

Small exploratory analysis of football match data for getting to know pandas.

The notebook focuses on:

- loading and inspecting match data
- selecting basic match, score, and card columns
- joining historical Elo ratings without using future information
- checking a few simple relationships around cards, Elo, and match results
- building a first baseline prediction model

This is intentionally a lightweight practice notebook, not a finished football prediction project.

## Files

- `analysis.ipynb`: main exploratory notebook
- `data/`: local dataset files, not necessarily included in this repository

## Dataset

The analysis uses the Club Football Match Data dataset by Adam Gabor. Match results and statistics are based on Football-Data.co.uk, and Elo ratings are based on ClubElo.

If you use the data, cite the original dataset:

Gabor, A. (2025). *Club Football Match Data*. Retrieved from https://github.com/xgabora/Club-Football-Match-Data-2000-2025/.
