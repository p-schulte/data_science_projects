# Data Science & Machine Learning Projects

## Installation

### 1. Create the Conda Environment
Create a Conda environment with all required dependencies:

```bash
conda create -n ml_ds python=3.10 \
numpy pandas scipy \
scikit-learn statsmodels \
matplotlib seaborn plotly \
jupyterlab notebook ipykernel \
tqdm joblib \
xgboost lightgbm catboost \
optuna shap mlflow \
imbalanced-learn \
openpyxl xlrd pyarrow fastparquet \
requests beautifulsoup4 lxml \
sqlalchemy psycopg2 \
black isort flake8 mypy \
conda-forge::nodejs \
-c conda-forge
```

### 2. Activate the Environment
```bash
conda activate ml_ds
```

### 3. Run the Notebooks
Once activated, the environment is ready to run the notebooks.

> **Note:** Development and execution are done using **VS Code** with Jupyter notebook support.
